ABOUT

Hi, Saya Krisna. Saya pembuat blog ini, tujuan saya membuat blog ini untuk mengshare cerita tentang saya semasa sekolah dari TK hingga Kuliah saat ini. Sebelum saya mengshare cerita-cerita menarik tentang diri saya ijin kan saya memperkenalkan diri.

Perkenalkan Nama Saya Maheda Krisna Mukti, Saya lahir di Brebes, Jawa Tengah pada tanggal 7 Agustus 2001. Umur saya saat ini 20 tahun dan saya anak pertama dari empat bersaudara. Hobi saya bermain game dan olahraga seperti basket, bola, dan badminton. Cita-cita saya yaitu ingin menjadi polisi seperti ayah saya yang bekerja di kepolisian namun mungkin takdir berkata lain dan akhirnya saya melanjutkan kuliah di Universitas Dian Nuswantoro Semarang dan Termasuk di Fakultas Ilmu Komputer Prodi Teknik Informatika..
